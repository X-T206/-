## 1.axis=0表示按列求和
```python
import torch
a=torch.tensor([[2,3,4],[5,3,1],[7,1,2]])
result=torch.sum(a,axis=0)
result
[out]:
tensor([14,  7,  7])
```
## 2.axis=1表示按行求和
```python
import torch
a=torch.tensor([[2,3,4],[5,3,1],[7,1,2]])
result=torch.sum(a,axis=1)
result
[out]:tensor([ 9,  9, 10])
```
##### N维矩阵特定轴求和也如上所讲，例3维矩阵shape=(D0,D1,D2)=(2,5,4)
* 1.axis=0:[5,4]
* 2.axis=1:[2,4]
* 3.axis=2:[2,5]
###### 以上情况都基于keepdims=False的情况，当keepdims=True时：
```python
import torch
a=torch.ones((2,5,4))
a.sum(axis=1,keepdims=True).shape
[out]:torch.Size([2, 1, 4])
```
